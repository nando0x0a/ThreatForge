  `require all denied` to NOT be set on affected directory aliases.
- **Observable Behaviour Patterns (in absence of specific IoCs)**:

  - POST requests to CGI-bin paths containing traversal sequences with shell command payloads in body
  - Anomalous outbound connections from Apache process (reverse shell indicators)
  - Apache spawning unexpected child processes (e.g., `/bin/sh`, `curl`, `wget`)
  - File creation in web-accessible directories by Apache process user (www-data)
