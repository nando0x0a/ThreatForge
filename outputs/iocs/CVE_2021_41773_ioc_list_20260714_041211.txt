## Observable Behaviour Patterns
# (Supplementary to IoCs above, for detection logic development)

1. **HTTP POST to CGI paths with traversal sequences** — Exploitation for RCE
   requires a POST request body containing shell commands (e.g., `echo Content-Type: text/plain; echo; id`) delivered to traversal URI.

2. **Anomalous HTTP response sizes on 200 responses to traversal URIs** —
   Successful /etc/passwd reads return predictable multi-line content.

3. **Execution of system binaries from Apache worker process** — Monitor for
   `httpd` or `apache2` spawning child processes such as `sh`, `bash`, `id`,
   `whoami`, `curl`, `wget`, `python`.

4. **Outbound connections from web server process** — Post-RCE reverse shell
   or payload retrieval; unexpected outbound TCP from Apache worker PID.

5. **Mass scanning pattern** — High volume of 400/403/404 responses to
   traversal URI patterns from single source IP; characteristic of automated
   exploitation frameworks targeting CVE-2021-41773 at scale.
