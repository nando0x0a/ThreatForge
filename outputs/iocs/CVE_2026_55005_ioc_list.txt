### Why IoCs Are Limited
- CVE-2026-55005 is **5 days old** as of generation date — insufficient time for OSINT-observable exploitation campaigns to be documented and attributed.
- No public PoC exists [1], reducing the likelihood that broad scanning/exploitation activity is currently detectable at scale.
- The MSRC advisory page [2] returned JavaScript-rendered content without structured IoC data.
- The vulnerability requires **attacker authorization** [1][3], meaning exploitation is unlikely to generate the same high-volume indiscriminate network noise as unauthenticated vulnerabilities. This narrows the detection surface to authenticated sessions and post-auth behaviour.

### Observable Behaviour Patterns (In Lieu of Confirmed IoCs)
Given the heap-based buffer overflow in Microsoft Exchange Server with a CVSS 8.8 (HIGH) [1] and network-based RCE vector, analysts should monitor for:

1. **Authenticated Exchange sessions followed by anomalous process spawning** — e.g., `w3wp.exe` or `UMWorkerProcess.exe` spawning unexpected child processes (cmd.exe, powershell.exe, wscript.exe).
2. **Memory anomalies in Exchange worker processes** — crash dumps, WER reports, or application event log errors in Exchange transport/backend services.
3. **Unusual outbound connections from Exchange servers** — particularly from IIS application pools or Exchange service accounts to non-Exchange infrastructure.
4. **Lateral movement from Exchange server host** — Exchange servers are high-value pivot points; monitor for unusual SMB, WMI, or RPC activity originating from Exchange hosts post-authentication.
5. **Exploitation of Exchange service accounts** — review for privilege escalation attempts or token manipulation following authenticated Exchange API calls.

### Recommended Actions
- Apply Microsoft security update per [2] immediately given CVSS 8.8 and T1/HIGH priority tagging.
- Baseline normal Exchange process trees and network egress **now**, before exploitation activity is observed.
- Re-evaluate this IoC list when PoC code or threat intelligence reporting becomes available.
