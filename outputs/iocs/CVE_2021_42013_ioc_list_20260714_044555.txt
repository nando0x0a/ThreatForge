### Affected Versions
- Apache HTTP Server 2.4.49 and 2.4.50 ONLY
- CVE-2021-42013 specifically resolves incomplete patch from CVE-2021-41773
- Versions ≤ 2.4.48 and ≥ 2.4.51 are NOT affected by this specific CVE

### Exploitation Conditions
- Requires mod_cgi or mod_cgid enabled for RCE
- Requires directory traversal outside Alias scope without `Require all denied`
- Path traversal (read-only) possible without CGI

### Analyst Actions Required
1. Validate URI pattern indicators against your environment's Apache access logs
2. Enrich IP indicators via GreyNoise, Shodan, and internal SIEM telemetry
3. Confirm Apache version on all internet-facing instances (patch to ≥ 2.4.51)
4. Hunt for webshells in Apache DocumentRoot and /tmp post-exploitation
5. Do NOT deploy these indicators to blocking rules without analyst review — URI
   patterns may produce false positives on legitimate double-encoded requests

### CISA KEV Obligation
This vulnerability is listed on the CISA Known Exploited Vulnerabilities catalogue.
US Federal agencies are subject to BOD 22-01 remediation deadlines.
All organisations should treat this as Priority 1 for patching.
