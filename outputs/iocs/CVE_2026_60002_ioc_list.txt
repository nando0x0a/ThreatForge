## Network Indicators

*No specific malicious IPs identified in provided context.*

**Observable Network Behaviour Patterns (for detection/hunting use):**

- Unexpected or repeated SSH key re-exchange (KEX) handshakes from a single client source, particularly where the server host key changes mid-session [2][3]
- SSH connections to or from OpenSSH servers running versions < 10.4 [2]
- Anomalous SSH session terminations or client-side crashes following key re-exchange events, which may indicate memory corruption attempts [1][4]
- Elevated frequency of SSH re-key negotiation (`SSH_MSG_KEXINIT`) messages within a single session, which could indicate a probing pattern for this vulnerability
