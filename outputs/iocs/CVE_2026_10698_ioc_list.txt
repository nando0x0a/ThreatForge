## Notes

### Vulnerability Context
- **CVE:** CVE-2026-10698
- **Affected Product:** Progress MOVEit Transfer versions 2025.0.0–2025.0.7, 2025.1.0–2025.1.3, 2026.0.0 [1][2][3]
- **Vulnerability Class:** SQL injection via improper neutralization of special elements in Custom Reports module data query logic [1][3]
- **CVSS Score:** 7.2 HIGH [1]
- **PoC Status:** No public proof-of-concept known at time of generation. Exploitation complexity may be lower once PoC becomes available given the nature of SQL injection classes.
- **KEV Status:** Not listed in CISA KEV at time of generation.
- **Age:** 6 days old — early in disclosure lifecycle; IoC availability is expected to be limited.

### Observable Behaviour Patterns (in absence of specific IoCs)

Since no confirmed exploitation IoCs are available, analysts should monitor for the following behavioural patterns in MOVEit Transfer logs and network telemetry:

1. **Anomalous SQL metacharacter sequences** in HTTP request parameters destined for Custom Reports endpoints — look for characters/sequences such as `'`, `--`, `;`, `UNION SELECT`, `OR 1=1`, `WAITFOR DELAY`, `SLEEP()`, `xp_cmdshell` in URL parameters or POST body data.

2. **Unusual query response timing** from the MOVEit Transfer application server that may indicate time-based blind SQL injection attempts (e.g., repeated requests with abnormal latency spikes).

3. **Elevated or unexpected database query volumes** originating from the Custom Reports module, visible in database audit logs.

4. **Authentication anomalies:** CVSS 7.2 may imply authenticated access is required [1]; watch for credential stuffing, unusual account enumeration, or access from unexpected geolocations prior to Custom Reports access.

5. **Bulk data exfiltration patterns:** SQL injection in a file transfer platform is high-risk for data theft — monitor for unusual data egress volumes from the MOVEit Transfer host.

6. **Requests from scanning infrastructure** to Custom Reports endpoints, particularly from known vulnerability scanner ASNs, given the 6-day age of this disclosure.
