## Attack Vector

### Vulnerability Mechanism

CVE-2025-59287 is a **deserialization of untrusted data** vulnerability in the Windows Server Update Services (WSUS) service. WSUS hosts a web-based service endpoint (typically over HTTP/HTTPS) that processes serialized .NET objects. The vulnerability arises when the WSUS service deserializes attacker-controlled data without adequate type validation or integrity checking, allowing the attacker to construct a malicious serialized payload that triggers arbitrary code execution in the context of the WSUS service process.

### Network Path and Conditions

- **Default WSUS Ports:** TCP 8530 (HTTP) and TCP 8531 (HTTPS) for client-facing traffic; TCP 80/443 where IIS-hosted configurations apply.
- **Attack surface:** Any host with network access to the WSUS listening port. In enterprise environments, WSUS servers often have broad inbound access from managed workstations and servers, significantly widening the blast radius.
- **Authentication bypass:** The unauthenticated nature (PR:N) means perimeter firewall rules permitting WSUS traffic are the only pre-existing control likely to block exploitation.
- **Typical pre-conditions:** None confirmed beyond network reachability. Specific vulnerable endpoint paths are not publicly confirmed for this CVE — see Data Note above.

### Endpoint / Process Telemetry

Successful exploitation of a .NET deserialization RCE in a WSUS (IIS/WCF-hosted) context typically produces:

w3wp.exe (WSUS App Pool)
  └─ cmd.exe or powershell.exe        ← unexpected child of w3wp.exe — HIGH SIGNAL
       └─ net.exe / whoami.exe        ← reconnaissance
       └─ certutil.exe / curl.exe     ← payload staging
       └─ regsvr32.exe / mshta.exe    ← LOLBin execution

**Key process chain alert conditions:**
- `w3wp.exe` spawning `cmd.exe`, `powershell.exe`, `wscript.exe`, or `mshta.exe`
- `UpdateServicesDbSetup.exe` or WSUS service binary (`wsuspool`) spawning unexpected children
- Outbound network connections from `w3wp.exe` (WSUS app pool identity) to non-Microsoft IP ranges
- New scheduled tasks or services created in close temporal proximity to anomalous WSUS traffic
